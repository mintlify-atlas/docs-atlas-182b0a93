Pipedream offers several support channels depending on your plan and the type of help you need.

## Community forum

The [Pipedream community forum](https://pipedream.com/community) is the primary place to ask questions, share workflows, and get answers from both the Pipedream team and other developers.

The forum is organized into sections:

- **[General questions](https://pipedream.com/community)** — how-to questions and troubleshooting
- **[Show and tell](https://pipedream.com/community/c/show-tell/8)** — share workflows and projects
- **[Feature requests](https://pipedream.com/community/c/feature-requests/7)** — suggest and vote on new features
- **[Bug reports](https://pipedream.com/community/c/bugs/9)** — report issues with the platform
- **[Developer / components](https://pipedream.com/community/c/dev/11)** — discuss component development

## Slack

Join the Pipedream Slack community for real-time discussion:

[pipedream.com/support](https://pipedream.com/support)

## GitHub

Use GitHub issues for bugs, feature requests, and app integration requests in the open-source repo:

[github.com/PipedreamHQ/pipedream/issues](https://github.com/PipedreamHQ/pipedream/issues)

Before opening a new issue, search existing issues to avoid duplicates. If an issue already exists, add a reaction or leave a comment with your specific use case.

<AccordionGroup>
  <Accordion title="Reporting a bug">
    Open a [bug report](https://github.com/PipedreamHQ/pipedream/issues/new?template=bug_report.md) with:

    - A clear description of the problem
    - Steps to reproduce
    - Expected vs. actual behavior
    - Screenshots if applicable
  </Accordion>
  <Accordion title="Requesting a feature">
    Open a [feature request](https://github.com/PipedreamHQ/pipedream/issues/new?template=feature_request.md) with:

    - The problem you are trying to solve
    - The solution you would like
    - Whether you have a current workaround
    - Any comparable features in other tools
  </Accordion>
  <Accordion title="Requesting an app integration">
    Open an [app integration request](https://github.com/PipedreamHQ/pipedream/issues/new?assignees=&labels=app%2C+enhancement&template=app---service-integration.md&title=%5BAPP%5D) with the app name and a link to its developer documentation.
  </Accordion>
</AccordionGroup>

## Email support

| Contact | Use for |
|---|---|
| [support@pipedream.com](mailto:support@pipedream.com) | General support and paid plan inquiries |
| [security@pipedream.com](mailto:security@pipedream.com) | Security vulnerability reports |

<Warning>
Do not report security vulnerabilities through public GitHub issues. Email [security@pipedream.com](mailto:security@pipedream.com) directly.
</Warning>
