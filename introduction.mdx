## What is Pipedream?

Pipedream is an integration platform for developers. It provides a hosted platform for connecting apps and developing event-driven automations — free for individual developers.

With Pipedream, you can:

- **Build workflows** that run automations triggered by HTTP requests, timers, or events from any connected app
- **Use pre-built components** from 2,800+ integrations to perform common actions without writing boilerplate
- **Write custom code** in Node.js, Python, Go, or Bash when you need logic that pre-built components don't cover
- **Embed integrations** directly in your product with [Pipedream Connect](/connect/overview)
- **Connect AI agents** to 2,800+ APIs with the built-in [MCP server](/connect/mcp/overview)

## How It Works

<Steps>
  <Step title="Create a trigger">
    Start a workflow with an HTTP endpoint, a scheduled timer, or an event source that watches for new data in an app like GitHub, Slack, or Google Sheets.
  </Step>
  <Step title="Add steps">
    Chain together pre-built action components or write your own Node.js, Python, Go, or Bash code. Each step can access data from previous steps via `steps`.
  </Step>
  <Step title="Deploy and run">
    Pipedream hosts and runs your workflow. You get real-time logs, error alerting, and a full event history with every input and output inspectable in the UI.
  </Step>
</Steps>

## Supported Languages

Write custom code in any of these runtimes within a workflow step:

- **Node.js** — Full access to `npm` packages. Import any package by declaring it in your code.
- **Python** — Full access to `pip` packages. Import any package and Pipedream installs it automatically.
- **Go** — Run Go code with access to Go modules.
- **Bash** — Shell scripts with access to common Unix utilities.
